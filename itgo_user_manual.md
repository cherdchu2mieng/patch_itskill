# 🛡️ IT-Skill itgo: User Manual & Quick Guide (v1.6.3)

คู่มือการใช้งานระบบจัดการกองเรืออธิปไตย (Sovereign Fleet Manager) สำหรับทีม IT

---

## 1. การเริ่มต้นใช้งาน (Bootstrap)

เพื่อให้เครื่องมือ `it-skill` สามารถเรียกใช้งานได้จากทุกที่ใน Terminal คุณต้องทำการเชื่อมต่อพาธ (PATH) เสียก่อน

### คำสั่ง Bootstrap
```bash
it-skill itgo bootstrap
```
- **สิ่งที่ระบบทำ**: ตรวจสอบและเพิ่ม `export PATH="$HOME/.it-skill/bin:$PATH"` ลงใน `.zshrc` หรือ `.bashrc` ของคุณโดยอัตโนมัติ
- **ตัวอย่างผลลัพธ์**: `✅ Aligned PATH in /home/user/.zshrc`
- *หมายเหตุ: หลังจากรันคำสั่งนี้ ให้ทำการรีสตาร์ท Terminal หรือรัน `source ~/.zshrc`*

## 2. การเรียกดูสถานะกองเรือ (Dashboard)

ใช้สำหรับตรวจสอบสุขภาพของเครื่องมือและดูรายการทักษะ (Skills) ที่ติดตั้งอยู่ในแต่ละ Agent

### คำสั่ง Dashboard
```bash
it-skill itgo
```
- **ข้อมูลที่แสดง**:
    - **Fleet Environment**: สถานะความพร้อมของ `it-skill` และเครื่องมือ Legacy Arra
    - **Skill Table**: รายการทักษะ เรียงลำดับโดยเอาทักษะทีม IT ไว้บนสุด
    - **Usage Stats**: จำนวนครั้งที่ทักษะถูกเรียกใช้งานจริง (Unique Sessions)
    - **Source-Aware Version**: เลขเวอร์ชันที่ดึงจากไฟล์ต้นฉบับจริง

## 3. การติดตั้งทักษะ (Sovereign Installation)

ติดตั้งกลุ่มทักษะตามโปรไฟล์ที่กำหนดลงในทุก AI Agent ที่ตรวจพบ (Claude, Codex, Gemini)

### คำสั่งติดตั้ง Standard Profile
```bash
it-skill itgo standard
```
- **ตัวเลือกเพิ่มเติม**:
    - `-y`: ข้ามขั้นตอนการยืนยัน (Auto-confirm)
    - `-g`: ติดตั้งแบบ Global (User-level)
- **ตัวอย่าง**: `it-skill itgo standard -y` (ติดตั้งทักษะมาตรฐาน IT ทั้งหมดทันที)

---

## 4. การซ่อมแซมทักษะรายตัว (Individual Refresh)

ในกรณีที่ทักษะบางตัวทำงานผิดปกติ หรือต้องการอัปเดตโค้ดเฉพาะตัวนั้นๆ โดยไม่กระทบทักษะอื่น

### คำสั่ง Refresh
```bash
it-skill itgo refresh <skill-name>
```
- **ตัวอย่าง**: `it-skill itgo refresh build-rfc -y`
- **สิ่งที่ระบบทำ**: ลบทักษะเดิมใน Agent ออก และคัดลอกเวอร์ชันล่าสุดจาก Source ลงไปใหม่ พร้อมอัปเดตดัชนี (Manifest) ทันที

## 🛡️ มาตรฐานความปลอดภัย (Safety Standards)
- **Auto-Backup**: ทุกครั้งที่คุณรัน `install` หรือ `refresh` ระบบจะทำการ Snapshot ทักษะเดิมเก็บไว้ใน `~/.config/it-skill/backups/` อัตโนมัติ
- **Domain Isolation**: ทักษะที่จัดการโดยทีม IT จะมีเครื่องหมาย `[IT]` กำกับชัดเจน และได้รับการคุ้มครองสิทธิ์ (Sovereign Guard) ไม่ให้ถูกเครื่องมืออื่นเขียนทับ
