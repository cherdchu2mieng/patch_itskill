# Change Request Detail: CR-006.v2 - Source-Aware Rich Manifest (Refinement)

## 1. CR Information
- **Parent RFC**: RFC-20260606-IT-GO-GENESIS-V1
- **Target Module**: it-skill-cli (cli/installer, cli/commands/itgo)
- **Status**: Pending

## 2. Technical Scope
- **Nature of Change**: Core Logic Alignment
- **Affected Components**:
    - `src/cli/installer.ts`: `installSkills` function
    - `src/cli/commands/itgo.ts`: Dashboard rendering logic
- **Logic Description**:
    1. **Installer Mapping**: ปรับปรุงฟังก์ชัน `installSkills` ให้ใช้ `skill.version` (ที่ได้จากระบบสแกนกลางใน CR-001) แทนการใช้ `pkg.version`
    2. **Rich Manifest Storage**: บันทึก Object `{ name, version }` ลงใน `.it-skill.json` เพื่อให้ข้อมูลถูกต้องตามต้นฉบับ
    3. **Dashboard Optimization**: ปรับปรุง Dashboard ให้สแกนไฟล์ดัชนี JSON เป็นหลัก (Fast Path) และใช้ข้อมูลเวอร์ชันที่เก็บไว้มาแสดงผลทันที

## 3. Impact Assessment
- **Integration Impact**: เพิ่มความน่าเชื่อถือให้กับระบบรายงานผล (Reporting)
